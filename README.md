A full-stack AI-powered campus waste management system with real-time deep learning classification, gamification, and GPS validation.

- Designed the complete **PostgreSQL database** — user records, disposal history, eco-points, leaderboard, and reward redemption via **SQLAlchemy ORM + psycopg2**
- Implemented **GPS-based bin proximity validation** at 5-metre accuracy and duplicate scan detection for data integrity
- Led a **4-member cross-functional team** (AI/ML · Frontend · Backend) using modular Flask Blueprint architecture

`Python` `Flask` `PostgreSQL` `SQLAlchemy` `Deep Learning` `REST API`

- Built an **AI eligibility engine** using scikit-learn and added **multilingual voice input** (English, Hindi, Chhattisgarhi) via Web Speech API

`Python` `Flask` `BeautifulSoup4` `scikit-learn` `Tesseract OCR` `Chrome Extension`
